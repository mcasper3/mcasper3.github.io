Music Voice

Music Voice is an Android application I am currently working on. The goal of this project is to produce an application that allows for hands-free Spotify use. This will be accomplished mostly through voice detection and hand gestures.

At the moment, the application is able to play music using the spotify API through the use of a GUI. I will be starting on implementing gesture recognition soon and voice control after that.
